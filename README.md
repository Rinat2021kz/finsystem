# ФинУчёт — управленческая финансовая система для малого бизнеса

Система управленческого учёта для малого бизнеса Казахстана (валюта — тенге).
Владелец вносит простые операции (доход / расход / перевод), система сама считает
ДДС, ОПУ, баланс денег и показывает понятный дашборд со светофором — без формул и Excel-ошибок.

Полное ТЗ — [docs/SPEC.md](docs/SPEC.md), схема БД — [docs/schema.sql](docs/schema.sql), правила разработки — [CLAUDE.md](CLAUDE.md).

## Стек

- **Next.js 15 (App Router) + TypeScript** — фронтенд и бэкенд в одном репозитории
- **PostgreSQL + Prisma** — данные (деньги — только BIGINT в тиынах)
- **Auth.js (next-auth v5)** — вход по e-mail/паролю, роли в `company_members`
- **Vitest** — юнит-тесты расчётного модуля (`lib/calc/`) по приёмочным сценариям SPEC
- **ExcelJS** — экспорт отчётов; PDF — печать браузера (MVP)

## Запуск локально

```bash
npm install
cp .env.example .env      # заполнить DATABASE_URL и AUTH_SECRET
npx prisma migrate deploy # применить миграции (или prisma db push)
npm run dev               # http://localhost:3000
```

Тесты расчётного модуля (приёмочные сценарии 1–4, 7, 10 из SPEC):

```bash
npm test
```

## Деплой (Vercel + Neon)

1. Создать бесплатную PostgreSQL-базу на [neon.tech](https://neon.tech) → скопировать `DATABASE_URL`.
2. Импортировать этот репозиторий на [vercel.com/new](https://vercel.com/new).
3. В настройках проекта Vercel добавить переменные окружения:
   - `DATABASE_URL` — строка подключения Neon;
   - `AUTH_SECRET` — `npx auth secret` или любые 32+ случайных символа;
   - `AUTH_TRUST_HOST` — `true`.
4. Применить миграции к базе: локально `DATABASE_URL=... npx prisma migrate deploy`.
5. Deploy — приложение доступно по адресу `https://<проект>.vercel.app`.

## Что уже реализовано (MVP, Фаза 1)

- Регистрация владельца, онбординг-мастер с отраслевыми шаблонами (6 отраслей)
- Справочники: счета, категории (иерархия + группы ОПУ), контрагенты
- Операции: доход / расход / перевод, фильтры, валидация (zod), запрет отрицательных сумм
- Расчётный модуль с тестами: ДДС (по `date_cashflow`), ОПУ (по `period_pnl`), баланс денег
- Отчёты по выбранному периоду, закрытие/открытие месяца со снимком и журналом аудита
- Дашборд со светофором и «нет данных» вместо ошибок
- Экспорт ДДС/ОПУ/баланса в Excel, печать в PDF
- Мультитенантность: каждая выборка — по `company_id` + проверка членства
